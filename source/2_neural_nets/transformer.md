## Attention and Transformers

### Attention mechanisms

Attention mechanisms are a key component of Transformers. They allow the model to focus on different parts of the input sequence.
We will learn about the following attention mechanisms:
- Self-attention
- Multi-head attention
- Attention is all you need

[Visualizing Neural Machine Translation Mechanics of Seq2Seq Models with Attention](https://jalammar.github.io/visualizing-neural-machine-translation-mechanics-of-seq2seq-models-with-attention/) is a great resource to understand the attention mechanisms.
