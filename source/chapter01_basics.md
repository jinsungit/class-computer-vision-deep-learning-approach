## Math basics


- **Calculus:** gradients, derivatives, min/max
- **Linear algebra:** vectors, matrices, operations
- **Probability:** random variables, distributions, sampling, Bayes

*Reference:* [Mathematics for Machine Learning](https://mml-book.github.io/book/mml-book.pdf)

## Machine learning basics


- **Supervised learning:** classification, regression, loss functions
- **Unsupervised learning:** clustering, dimensionality reduction
- **Model evaluation:** train/validation/test splits, metrics, overfitting

*Reference:* Pattern Recognition and Machine Learning (Bishop)

## Deep learning basics


- **Neural networks:** perceptron, MLP, activation functions
- **Backpropagation:** chain rule, gradient computation
- **Architectures:** feedforward, residual connections, attention

*Reference:* [Deep Learning (Goodfellow et al.)](https://www.deeplearningbook.org/)

## Optimization


- **Gradient descent:** SGD, mini-batch
- **Optimizers:** Adam, AdamW, momentum, learning rate scheduling
- **Regularization:** dropout, weight decay, batch normalization

*Reference:* [D2L – Optimization for Deep Learning](https://d2l.ai/chapter_optimization/index.html)
