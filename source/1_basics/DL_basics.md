## Neural networks


Deep learning is a subfield of machine learning that uses neural networks to learn representations of data.

Note that we are no longer drawing inspiration from biology to build modern neural network models.

## Perceptron

*definition*:
In the context of neural networks, a perceptron is an artificial neuron using the Heaviside step function as the activation function.

--Wikipedia
